## Usage

### CHIRIMEN (browser)

```html
<script src="https://cdn.jsdelivr.net/npm/@chirimen/adt7410"></script>
```

Using ES Modules:

```html
<script type="module">
  import ADT7410 from "https://esm.run/@chirimen/adt7410";
</script>
```

### Node.js

```js
const ADT7410 = require("@chirimen/adt7410");
```

Using ES Modules:

```js
import ADT7410 from "@chirimen/adt7410";
```
